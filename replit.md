# 多言語対話AIプラットフォーム

## プロジェクト概要
VoiceVoxテキスト読み上げ技術を活用したアニメ風キャラクター対話アプリケーション。各キャラクターは独自のペルソナ、音声、視覚表現を持ち、性格特性に基づいて適切な話し方や代名詞を使用して応答します。

## 最新の変更 (2025年7月8日)
✓ **Claude AI統合完了**: 話者B（右側キャラクター）専用でClaude API実装
✓ **フォールバック機能**: Claude APIクレジット不足時は自動的にGPT-4.1にフォールバック
✓ **話者A/B分離**: 話者A（左側）はGPT-4.1、話者B（右側）はClaude AI使用
✓ **多様なAIエンジン**: 異なるAIモデルによる多様な会話スタイル実現
✓ **会話文脈保持**: Claude実装でも既存の文脈認識機能を維持

## プロジェクト構成
- **Flask**: Webフレームワーク
- **OpenAI GPT-4.1**: 話者A用AIエンジン
- **Anthropic Claude**: 話者B用AIエンジン（フォールバック機能付き）
- **VoiceVox**: 音声合成API
- **JavaScript**: インタラクティブUI

## キャラクター設定
各キャラクターは独自の性格、話し方、代名詞使用パターンを持つ：
- **四国めたん**: 一人称「ワタシ」、二人称「アンタ」、ツンデレ、中二病
- **春日部つむぎ**: 一人称「私」、二人称「きみ」、優等生、丁寧語
- **雨晴はう**: 一人称「僕」、二人称「あなた」、元気、関西弁
- **WhiteCUL**: 一人称「私」、二人称「あなた」、冷静、知的

## 技術仕様
- **話者ポジション判定**: キャラクターUUIDに基づく自動AI選択
- **会話履歴管理**: セッションベースの会話継続
- **文脈認識**: 直前の話題や流れを分析して自然な応答生成
- **エラーハンドリング**: APIエラー時のフォールバック機能

## ユーザー設定
- 自然な日本語での対話を重視
- キャラクター間の関係性と呼称の一貫性維持
- リアルタイムでの音声・視覚フィードバック
